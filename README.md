# Sobre o projecto template - Tema Khoweweb Blogger

O **Khoweweb Blogger** é um tema profissional desenvolvido em **XML nativo do Google Blogger**, integrando **HTML5, CSS3 e JavaScript** num único ficheiro, conforme o padrão exigido pela plataforma. Este tema é de uso ideal para Websites institucionais, 

O tema foi concebido para a criação de websites modernos, responsivos e otimizados para SEO, direcionado a negócios, Blogs profissionais, Portfólios, projetos editoriais, Revistas digitais e Landing pages no Blogger este tema se diferência dos outros pela existência de um programa de referência e pela disponibilidade de métodos de pagamento disponíveis no website.

## Estrutura do Projeto

### Tipo de Projeto
Tema XML completo para Google Blogger

Ficheiro único: `KhodewebBlogger_Final_v6.xml`

### Organização Interna
Utilização extensiva de: `<b:section>` `<b:widget>` `<b:include>` `<b:if>`

Separação lógica entre: `Estrutura HTML` `Estilos CSS` `Scripts JavaScript` `Lógicas Condicionais` 

Fácil manuntabilidade: `Código comentado` `Estrutura modular` `cores e fontes editaveis pelo personalizador Blogger`

## Tecnologias Utilizadas:  
Tecnologias: `Google Blogger` `XML` `HTML5` `CSS3` `JavaScript (Vanilla)` `Google Fonts` `Schema.org (dados estruturados)` `MailerLite` `Google Forms` `Google sheets`

Funcionalidades Implementadas: 
`Layout totalmente responsivo` `Sistema de navegação configurável` `Slider / secção de destaque` `Secções modulares controladas por widgets` `Área de testemunhos` `Integração com formulários externos` `Integração com newsletter (MailerLite)` `Sistema de comentários nativo do Blogger` `Estrutura otimizada para SEO` `Dados estruturados para motores de busca` `Widgets reutilizáveis` `documentação para manuntenção e escabilidade`

## Conclusão

O **Khoweweb Blogger** é um tema blogger robusto, completo e profissional, bem estruturado e tecnicamente sólido, explorando ao máximo as capacidades do Google Blogger mas matendo uma fácil instalação e configuração com código organizado para fácil escalábilidade, organização e com funcionalidade modernas.
